使用的时候请修改  `/src/page/index/index.vue` 中的 key，改成自己申请的 `appID`

```js
AMapLoader.load({
    key: "替换成自己的 应用ID", // 开发应用的 ID
    version: "2.0",   // 指定要加载的 JSAPI 的版本，缺省时默认为 1.4.15
```
